Visual Studio Code en PlatformIO

In deze sectie leer je hoe je Visual Studio Code (VS Code) kunt gebruiken in combinatie met PlatformIO om je microcontrollerprojecten te ontwikkelen. VS Code is een krachtige en veelzijdige code-editor die veel wordt gebruikt door ontwikkelaars over de hele wereld, terwijl PlatformIO een open-source ecosysteem is dat ondersteuning biedt voor verschillende embedded platforms en frameworks.

Installeren van Visual Studio Code

  1. Ga naar de officiële website van Visual Studio Code: https://code.visualstudio.com/

  2. Klik op de downloadknop en volg de instructies om VS Code te installeren op jouw besturingssysteem..

Installeren van PlatformIO Extension

  1. Open Visual Studio Code.

  2. Ga naar de Extensions Marketplace door op het vierkante icoon aan de linkerkant van het venster te klikken of gebruik de sneltoets Ctrl+Shift+X.

  3. Zoek naar “PlatformIO IDE” in de zoekbalk.

  4. Klik op de “Install” knop om de PlatformIO IDE-extensie te installeren.

Handleiding voor het gebruik van PlatformIO in VS Code

  1. Na het installeren van de PlatformIO-extensie, klik op het PlatformIO-icoon in de zijbalk van VS Code om het PlatformIO Home-scherm te openen.

  2. Klik op “New Project” om een nieuw project aan te maken.
